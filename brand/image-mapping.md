## File location

Source originals stay in `/brand/` (unrenamed, archived). Web-ready copies are placed in `/app/public/images/` with the semantic names above. This way:

- `/brand/` is the source-of-truth archive (originals)
- `/app/public/images/` is what the live site references
- Future content additions follow the same pattern
