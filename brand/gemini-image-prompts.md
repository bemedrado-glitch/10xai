# 10XAI  Gemini Nano Banana 2 Prompt Library

> Production-ready prompts for Gemini 2.5 Flash Image (Nano Banana 2). Every prompt locks the brand palette and excludes forbidden AI clichs. Paste into Google AI Studio, Gemini API, or the Gemini app  attach the new 3D logo (`logo-coin-3d.png`) as a style reference where noted to lock visual identity.
>
> Last updated: 2026-04-30

---

## How to use this library

1. **Always feed the new 3D logo as a reference image** when generating new brand assets  Nano Banana 2 will copy the brushed-gold + matte-black material treatment automatically. (Prompts marked  ★ benefit most.)
2. **Aspect ratios** are explicit in each prompt. Override only if the surface needs it.
3. **Palette is locked**: `#1C1917` ink, `#CA8A04` gold, `#FAFAF9` cream, optional `#FEF9C3` highlight. Never deviate.
4. **Hard "do not" list** repeated in every prompt for consistency: no robots, no humanoid AI, no neural-network glow, no purple/pink gradients, no glowing keyboards, no holographic UI, no cyan/electric blue, no stock-photo team-around-laptop tropes, no cartoon emojis as decoration.

## Tips for getting consistent results from Nano Banana 2

1. **Always paste the locked palette in hex.** Nano Banana 2 is good at hex matching when explicit; vague color words drift.
2. **Use "no" lists at the end of each prompt.** This model respects exclusions when explicitly stated. The boilerplate "no robots, no AI clichs, no purple/pink, no neon, no holograms" should appear in every prompt.
3. **Provide a reference image** of the existing 3D logo (`logo-coin-3d.png` or `logo-wordmark-3d.png`) when generating any "extends the brand" asset  it'll match material and lighting.
4. **Specify camera/lens/aperture** for photographic prompts. "Shot on 35mm at f/2.8, light film grain" produces more editorial, less AI-stock-photo results.
5. **Negative space is a feature, not a bug.** Ask for it explicitly  most generators over-fill compositions.
6. **Aspect ratios first**, before any other styling. Nano Banana 2 obeys explicit aspect-ratio requests when stated up front.
7. **Iterate by surgical edits.** If 80% of an image works, prompt with: *"Same composition, same palette, but replace [X] with [Y]"*  it preserves what worked.
